# Project Samples

A selection of code samples from past projects. One subfolder
per project, each with its own README summarizing what it does. Code only; no
participant/research data is included anywhere in this folder.

## Projects

- [`gps-otree-experiment/`](gps-otree-experiment/) — Python (oTree) implementation of an
  incentivized economic-preferences experiment (risk, time, trust, altruism, reciprocity)
  for the Global Preferences Survey, run across multiple countries.
- [`income-mentalhealth-ml-shap/`](income-mentalhealth-ml-shap/) — ML pipeline (XGBoost,
  Random Forest, Logistic Regression, etc.) predicting child income/mental-health outcomes
  from longitudinal survey data, with SHAP explainability, bootstrap validation, and
  Wilcoxon significance testing across models.
- [`immigrants-labor-market-study/`](immigrants-labor-market-study/) — survey data analysis (t-tests,
  Mann-Whitney, Mincer wage regressions) comparing early labor-market outcomes between immigrant
  and native university graduates in Germany, including a full `pytask`-based analysis package.
- [`immigrants-pay-premium-us/`](immigrants-pay-premium-us/) — cross-country analysis (US, UK,
  Canada, Australia, Germany) of the immigrant/native pay gap using cell-level average-pay data,
  with fixed-effects OLS regressions and a US-focused deep dive.
- [`investment-screening-agent/`](investment-screening-agent/) — LLM agent (Claude +
  pydantic-ai) that screens public companies against a growth-investing mandate, combining
  financial fundamentals and live news search into a structured recommendation, with a
  Streamlit UI.
